This week the focus is on how to compartmentalize your code into different folders and files.

The task this week is to:
1. Download this folder and run your respective script (main.cpp or main.py)
2. Edit and submit pull request changing your function

To run the python code:

python main.py

To run the cpp code:

g++ -c cpp_files/other_patrick.cpp cpp_files/other_mason.cpp
g++ other_patrick.o other_mason.o main.cpp -o include-other-files.exe
